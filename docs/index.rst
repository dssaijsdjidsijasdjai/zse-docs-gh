**Apple Collector**  
====================
Jest to program, służący do optymalizacji zbierania jabłek w sadzie. 

**Instalacja**
--------------------
Do zainstalowania programu, należy użyć komendy
``pip install -U AppleCollector``

Przykłady użycia:

* Kuracja pinginów
* Spawanie bananów
